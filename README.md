# Chi Cafe Site

Interactive static web app for a cafe brand. It includes an animated landing page, menu filters, a browser-saved order tray, loyalty stamp card, gallery lightbox, and reservation form.

## Run locally

Open `index.html` in a browser.

## Features

- animated hero and cafe mood picker
- menu search and category filters
- localStorage order tray
- loyalty stamp interaction
- gallery lightbox
- reservation/contact flow
